Vanishing-point coordinates are reported in canvas pixels from the top-left corner. The default cube centre is (0, 0, 0), size is 1, yaw is 0°, roll is 0°, cube pitch is 0°, and camera pitch is 0°. Moving the cube vertically no longer changes the horizon. Camera pitch controls the horizon height and creates the vertical vanishing point; camera height moves the eye without tilting it. Double-click any slider to reset it. Double-click the canvas to reset all sliders and centre the view without changing the fisheye checkbox. Field-of-view changes are distance-compensated so the cube remains approximately the same size on screen. Drag empty canvas space to pan, use the mouse wheel to zoom about world origin (0, 0), and drag a cube edge to rotate using the most screen-normal available yaw, roll, or pitch axis. Construction lines now stop at a visible vanishing point instead of running past it. Enable 5/6-point fisheye for a 180° circular projection with curved lines and separate positive and negative axis vanishing points. Toggle Guidelines to draw low-opacity coloured rays from each outer frame-grid intercept to the current vanishing points.
